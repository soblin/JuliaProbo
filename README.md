## JuliaProbo

Re-implementation of [『詳解　確率ロボティクス』](https://github.com/ryuichiueda/LNPR_BOOK_CODES) in Julia language．Place the dataset in `/sensor_data`.

### Dependency

- julia(1.5.3)
- jupyter(python3)
- seaborn(python3)

### Branch

Basically the package in branch `ch<n>` only works for `notebook/ch<n>*.ipynb`.
